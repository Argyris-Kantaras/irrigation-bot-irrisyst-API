Irrigation should be applied when soil moisture drops below 25%.
Clay soils retain water longer than sandy soils.
Weather forecasts should be checked before irrigation.
Drip irrigation is more efficient than sprinkler irrigation.
Distance between Control station and last valve 400-450m maximum you better use "Multicable system".
For a small system when the distance is not that long we can use multi cable.

Medium and Large Systems more than 500 meters long is recommended to use SingleNet or RadioNet
If you choose SingleNet then you definetely need a Host: 00035-00176--Singlenet Host
If there are not more than 2 valves in one group then you should use 74340-01490--RTU 2x2
00035-00176--Singlenet Host
74340-01490--SingleNet RTU 2x2
74340-015000--SingleNet RTU 4x4
If there are more than 2 valves and up to 4 in one group then you should use 74340-015000--RTU 4x4
If there are more than 4 valves in one group then you should combine RTUs to get the needed number.
Always in SingleNet system you should install SLSM for lighting protection and minimum 2pcs RTUs with PLSM
SLSM is needed in the pump station exactly after the Host.
If Netacap sensors are needed for the SingleNet system, then you should use GS-ONE units to the system.
Each GS ONE unit can control 1 Netacap and 2 Digital Inputs.
In SingleNet system you cannot install Netacap directly to it, You should add GS ONE if needed.
